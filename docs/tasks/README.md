## MCPツール一覧

実装する11種類のMCPツール：

1. **read_file** - ファイル内容を読み取る
2. **write_file** - ファイルに書き込む
3. **insert_code** - コードを挿入する
4. **delete_code** - コードを削除する
5. **replace_code** - コードを置換する
6. **execute_command** - ターミナルコマンドを実行する
7. **execute_build** - ビルドを実行する
8. **get_diagnostics** - 診断情報を取得する
9. **get_quick_fixes** - クイックフィックスを取得する
10. **apply_quick_fix** - クイックフィックスを適用する
11. **format_document** - ドキュメントをフォーマットする

## タスクの進め方

### 1. 各フェーズの開始前
- フェーズのタスクファイルを確認する
- 依存関係を理解する
- 必要なリソースを準備する

### 2. 日次タスクの実施
- 1日1タスクを基本とする
- 受け入れ基準をすべて満たす
- テストを作成する
- ドキュメントを更新する

### 3. 各フェーズの完了時
- 完了基準をすべて確認する
- レビューを実施する
- 次フェーズへの引き継ぎ事項を整理する
